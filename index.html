Vue.js

Ember


is an open-source JavaScript web framework, based on the Model–view–viewmodel (MVVM) pattern. It allows developers to create scalable single-page web applications by incorporating common idioms and best practices into the framework.

Git

GNU/Linux

PHP

MySQL
